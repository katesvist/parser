# Тендер.Поиск

Веб-приложение для парсинга и поиска тендеров с zakupki.gov.ru, аналогичное Контур.Закупки.

## Возможности

- 🔍 **Расширенный поиск тендеров** с множественными фильтрами
- 📊 **Интерактивный дашборд** с графиками и аналитикой
- ⭐ **Избранные тендеры** для быстрого доступа
- 💾 **Сохраненные поиски** с автоматическим отслеживанием
- 📄 **Детальный просмотр** всех данных тендера
- 👤 **Личный кабинет** для управления профилем

## Структура данных

Приложение работает с полной структурой данных из ЕИС zakupki.gov.ru:

### Основная таблица тендеров (tenders_gov)

- **Идентификаторы**: object_number, href, zakon
- **Общая информация**: object_info, placingway_name, placingway_code
- **Электронная площадка**: etp_name, etp_code, etp_url
- **Даты**: startdt, enddt, biddingdt, summarizingdt, contract_enddate
- **Финансы**: maxprice, currency_code, finance_total
- **Обеспечение**: part, procedureinfo, банковские реквизиты
- **Классификация**: okpd2info, ikz_code, kvr_code
- **Место поставки**: deliveryplace, garaddress, countrycode
- **Требования**: servicerequirement, manufacturerequirement, warrantytermdt
- **Заказчик**: fullname, shortname, inn, kpp, адреса, контакты

### Позиции закупки (tender_items)

- item_name, item_code
- okpdcode, okpdname
- quantity_value, quantity_name
- price_for_one, total_sum

### Документы и вложения (tender_attachments)

- file_name, doc_kind_name
- file_size, doc_date
- url для скачивания

## Фильтры поиска

- Закон (44-ФЗ, 223-ФЗ)
- Регион
- Статус тендера
- Способ закупки
- Код ОКПД2
- ИНН заказчика
- Диапазон цен
- Даты публикации и подачи заявок
- Номер извещения
- ИКЗ

## Технологии

- **React** с TypeScript
- **Tailwind CSS** для стилизации
- **Radix UI** компоненты (через shadcn/ui)
- **Recharts** для графиков и аналитики

## Особенности реализации

### Типизация данных

Все данные тендеров полностью типизированы в соответствии со структурой БД PostgreSQL. Используются интерфейсы `Tender`, `TenderItem`, `TenderAttachment`.

### Форматирование

Утилиты для форматирования данных:
- `formatCurrency()` - форматирование денежных сумм
- `formatDate()` - форматирование дат
- `formatFileSize()` - размер файлов
- `getStatusInfo()` - статус и цветовая схема

### Отображение данных

- **Поиск тендеров**: таблица с основной информацией
- **Детали тендера**: полная информация с вкладками
  - Позиции закупки
  - Требования
  - Документы
  - Дополнительная информация
- **Карточки**: заказчик, даты, площадка, обеспечение

## Данные для входа

Для демонстрации можно использовать любые данные - реальная авторизация не требуется.

## Примечания

Это frontend-приложение с моковыми данными. Для работы с реальной базой данных необходимо подключить backend API.
